# Spanish Colors

**Learn the colors in Spanish!**

This project provides a basic introduction to some common Spanish colors.

## Included Colors:
* RED: ROJO 
* BLUE: AZUL
* ORANGE: ANARANJADO
* GREEN: VERDE
* YELLOW: AMARILLO
